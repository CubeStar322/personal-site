Макет сторінки

JavaScript (часто скорочують як JS) — це мова програмування, що використовується для створення динамічного та інтерактивного контенту на веб-сторінках. Вона дозволяє розробникам реалізовувати різноманітні функції, такі як анімації, обробка подій, валідація форм, і багато іншого. JS може виконуватися на стороні клієнта (у браузері) та на стороні сервера (з використанням таких платформ, як Node.js).
CSS (Cascading Style Sheets) — це мова стилів, яка використовується для опису вигляду і форматування веб-сторінок. CSS дозволяє задавати кольори, шрифти, розміри, відступи, вирівнювання та інші візуальні характеристики HTML-елементів.
HTML (HyperText Markup Language) — це стандартна мова розмітки для створення веб-сторінок.HTML є основою веб-розробки, і він часто використовується разом з CSS і JavaScript для створення інтерактивних і стильних веб-сторінок.
C — це одна з найпопулярніших і найвпливовіших мов програмування, розроблена в 1970-х роках. Вона є мову низького рівня, що надає можливість програмістам працювати з апаратним забезпеченням та системними ресурсами.
C++ — це мова програмування, розроблена на основі C, яка додає підтримку об'єктно-орієнтованого програмування (ООП). Вона була створена в 1980-х роках і швидко стала популярною завдяки своїй гнучкості та потужності.
